This page explains the eligibility, submission methods, and maintenance principles for friend links on this site. Please read the complete rules before applying.

Last updated: July 22, 2026.

Basic Requirements

  • The website should be an accessible personal site or independent blog with a clear name and original content that is still being maintained.
  • Friend links are reciprocal. Add this site to your friend-links page before applying and make sure the link is publicly accessible.
  • It should use HTTPS and must not contain malicious redirects, scams, phishing, malware, or anything else that could put visitors at risk.
  • Sites centered on illegal material, explicit adult content, hate or harassment, piracy, advertising, copied content, or search-ranking manipulation will not be accepted.
  • Your subject, technology stack, traffic, and publishing schedule do not need to match this site. A genuine, complete, and maintained website is enough.

Information to Submit

Please include:

  1. Website name;
  2. Website URL;
  3. A one-line description;
  4. An avatar or site-icon URL;
  5. The URL of the friend-links page where you added this site;
  6. An English name and description, if available.

If no English name or description is supplied, the English friend-links page will preserve the original wording instead of inventing a translation.

How to Apply

Return to the Links page and submit publicly in the comments, or email mr.jingyuan.zheng@gmail.com.

Comments are visible to all visitors. Use email if your message contains anything you do not want to make public. Add this site before applying and include the public friend-links page where the link can be verified. This site will add your link after confirming it.

Review and Maintenance

  • Requests are reviewed manually. Meeting these rules does not guarantee acceptance, and there is no fixed response time.
  • After review, I will reply through the comment thread or email used for the request, whether it is accepted or declined. If declined, I will include the reason, so please watch for a response.
  • Names and descriptions normally use the applicant’s supplied wording. If the text is too long for the card, I may ask for a shorter version.
  • A link may be removed if the site remains unavailable, changes ownership, redirects elsewhere, or no longer meets these rules.
  • If the other site removes this site’s link without notice, its link may also be removed here.
  • You can request an update by comment or email when your URL, name, description, or avatar changes.
