Mac DMG 安装包怎么做？用 DMG Maker 自动生成带背景的安装镜像

一个简单的 Bash 脚本，用于生成带自定义背景和精确图标位置的 DMG 安装程序。

12/May/2026
阅读时长: 1 分钟
English

本文介绍 DMG Maker，一个帮 macOS 小工具自动生成 DMG 安装包的脚本，适合需要背景图、图标摆放和快捷链接的分发场景。

我刚刚开源了 DMG Maker，这是一个旨在简化 macOS 专业安装包制作过程的小工具。

为什么使用 DMG Maker？

手动制作带背景图的 DMG 通常需要编写复杂的 AppleScript 或者繁琐的 Finder 手动操作。DMG Maker 通过一行命令将这一切自动化。

核心功能

  • 自动软链接: 自动识别文件类型（如 .dictionary 或 .app）并链接到对应的系统目录。
  • 自定义视觉: 轻松应用背景图并根据坐标定位图标。
  • 开源: 欢迎在 GitHub 上查看源码。

快来尝试为你的项目制作精美的安装包吧！
